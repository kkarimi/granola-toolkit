# granola-toolkit docs

This directory is the dedicated documentation app for `granola-toolkit`.

It follows the same broad shape as the `emdash` repo:

- a separate `docs/` application
- content stored as MDX in `content/docs`
- a small landing page plus a structured docs tree

Run it locally with:

```bash
npm run dev
```

Or from the repo root:

```bash
npm run docs:dev
```
